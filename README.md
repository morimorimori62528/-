# 階層内ファイルリスト化ツール (File Hierarchy Lister)
「このフォルダにはどんなファイルがあるか」
そんな悩みを解決します。

## 主な特徴
直感操作（UIを通して）「調べたい場所」を指定。
一覧化した内容をExcel形式で出力します。

<br>**【全体把握】**<br>
- 行単位で全ファイル書き出し
- ファイルの中身をストレスフリーで確認可能（ハイパーリンク搭載）
<br>**【分析】**<br>
- フィルタ抽出（第三階層まで項目として記載）
- どのファイルが重たいか（ファイルのbyte数を記載）
- 新旧比較（ファイルの作成日時を記載）
- フォルダごとの総数（別シートに集計ファイルを追加）

## 技術スタック
- **言語**: Python 3.14
- **主要ライブラリ**:
  - `pandas` (データ加工・Excel出力)
  - `openpyxl` (ExcelWriter連携)
  - `tkinter` (GUI画面・プログレスバー)

## 使い方
 1. 「対象フォルダ」ラベルの「参照」ボタン 
      → 一覧化したいフォルダを指定
 2. 「保管先」ラベルの「参照」ボタン
      → 一覧化したExcelの出力先を指定
 3. 「実行」ボタン
      → 開始します。プログレスバーで進捗確認可能
 4. 「キャンセル」ボタン
      → 途中で中断可能
